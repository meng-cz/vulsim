# Vul 前后端交互操作文档

## 操作包格式

请求JSON格式：

```json
{
    "name": "operation_name",
    "args": {
        "key1": "value1",
        // 操作参数
    }
}
```

返回JSON格式：

```json
{
    "code": 0,
    "msg": "Success",
    "results": {
        "key1": "value1"
        // 操作返回数据
    },
    "list_results": {
        "key1": [
            "item1",
            "item2"
            // 操作返回的列表数据
        ]
    }
}
```

## load 操作

load 操作用于加载 Vul 项目文件及其相关模块、配置库和 Bundle 库。

### 请求参数

| 参数名         | 必填 | 描述                         |
|---------------|------|------------------------------|
| path          | 是   | Vul 项目文件 .vul 的路径        |
| import_paths  | 否   | 额外的导入模块路径列表，以 冒号 作为分隔符 |

## 返回结果

| 是否为列表 | 参数名         | 描述                         |
|----------|---------------|------------------------------|
| 是       | logs          | 加载过程中的日志信息列表            |
