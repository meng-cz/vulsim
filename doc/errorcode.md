# 错误码表

错误码为#开头的五位数字如`#11001`，高两位表示错误类型，后三位表示具体错误项

## 错误类型表

| 编号 | 描述 |
| --- | --- |
| #10 | 加载和解析项目文件错误 |
| #11 | 组件名检查错误，包括Bundle、Combine、Instance、Pipe、ConfigItem的名字检查 |
| #12 | 类型名检查错误，包括Bundle、Combine、Pipe中的各种变量类型检查 |
| #13 | Req-Serv连接检查错误 |
| #14 | Pipe连接检查错误 |
| #15 | 更新次序约束检查错误 |
| #16 | 保存项目文件错误 |
| #20 | 修改ConfigLib时错误 |
| #21 | 修改Bundle时错误 |
| #22 | 修改Combine时错误 |
| #23 | 修改Instance时错误 |
| #24 | 修改Pipe时错误 |
| #25 | 修改Connection时错误 |
