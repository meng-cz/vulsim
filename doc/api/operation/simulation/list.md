# simulation.list - 列出项目的仿真任务

## 概述

列出当前打开项目的所有仿真任务（run id）以及各任务在生成和编译阶段的完成时间戳信息。

## 参数

- 无

## 列表结果

- `runids` — 仿真任务 ID 列表，字符串数组。
- `gen_finished_us` — 每个任务生成完成时间（自纪元以来的微秒），未完成或不存在则为 `0` 或空字符串，字符串数组。
- `comp_finished_us` — 每个任务编译完成时间（自纪元以来的微秒），未完成或不存在则为 `0` 或空字符串，字符串数组。

## 错误码

- 无特殊错误码，若项目运行目录不可读则返回空结果。
