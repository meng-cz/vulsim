# code.update - 更新模块中指定代码块或接口的代码行

## 概述

更新项目中指定模块的某个代码块（user tick）、服务处理代码或子实例请求的 C++ 代码行。

## 参数

- `[0] module_name` — 要更新代码的模块名（字符串），必需。
- `[1] codeblock` — （可选）用户 tick 代码块名，若提供则更新该代码块的代码。
- `[2] service` — （可选）服务端口名，若提供则更新该服务的代码。
- `[3] instance` — （可选）子实例名称，若提供则与 `request` 一起用于定位请求代码。
- `[4] request` — （可选）子实例请求名称，若提供则与 `instance` 一起用于定位请求代码。
- `[5] code_lines` — 要更新的 C++ 代码行列表，使用换行符分隔的多行字符串。

## 结果

- 无（仅返回操作成功或错误信息）。

## 列表结果

- 无

## 错误码

- `EOPCodeGetInvalidArg` — 参数不合法（例如互斥参数同时提供或请求/实例参数不完整）。
- `EOPModCommonNotFound` — 指定模块未找到。
- `EOPModCommonImport` — 指定模块为导入模块，无法修改。
